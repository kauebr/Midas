## Sobre o Projeto

O MIDAS BR é um laboratório de pesquisa quantitativa desenvolvido em Python para investigar uma hipótese fundamental:

> É possível extrair padrões estatisticamente relevantes dos mercados financeiros utilizando Machine Learning?

O objetivo do projeto não é prometer previsões ou retornos financeiros.

O foco está em:

* Backtesting rigoroso
* Validação temporal
* Engenharia de atributos
* Avaliação estatística
* Reprodutibilidade científica
